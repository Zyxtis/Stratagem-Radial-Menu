Скрипт AutoHotkey для Helldivers 2, призванный изменить ваш стиль игры за счет автоматизации ввода стратагем, глубокой кастомизации и широкого набора функций.

Этот скрипт оптимизирует ввод стратагем благодаря автоматизации и настраиваемому Радиальному Меню. Вместо того чтобы заучивать сложные коды или забивать клавиатуру десятками биндов, вы можете вызывать любую стратагему через интуитивно понятный визуальный интерфейс.

### Основные возможности:

    Радиальное меню стратагем: Стильное экранное круговое меню, позволяющее быстро выбирать и активировать стратагемы при помощи мыши.
    Активация одной кнопкой: Назначьте вызов радиального меню на любую удобную клавишу клавиатуры или кнопку мыши.
    Выбор режима ввода: Поддержка стрелок, WASD или любой пользовательской раскладки для соответствия вашим игровым настройкам.
    Удобный интерфейс (GUI): Легко управляйте списком стратагем, профилями и настройками через графическое окно.
    Авто-пауза/Закрытие: Активируйте эту опцию для автоматического управления состоянием скрипта.
    Настройка стратагем: Добавляйте новые или редактируйте существующие стратагемы по мере развития игры.

### Инструкция по использованию:

1. Установка
   - Убедитесь, что у вас установлен AutoHotkey v2.
   - Распакуйте архив.
   - Запустите файл `Radial_menu.ahk`.
2. Начальная настройка
   - Откройте вкладку Settings (Настройки).
   - Назначьте клавишу вызова радиального меню — Radial Menu Key (По умолчанию: Средняя кнопка мыши).
   - Укажите клавишу вызова меню стратагем — Stratagem Menu Key, соответствующую вашей клавише в игре (По умолчанию: Left Ctrl).
   - Выберите раскладку ввода — Stratagem Input Layout (Настоятельно рекомендуются «Стрелки», так как игра по умолчанию использует WASD).
3. Управление биндами
   - Используйте интерфейс, чтобы добавить (+) или удалить (-) стратагемы для вашего активного профиля.
   - Двойной клик (ЛКМ) по стратагеме во всплывающем списке отметит её как «избранную». Это позволяет выделить важные стратагемы (например, Реинфорс или Ресаплай) и сортировать их одним кликом.
4. Использование радиального меню
   - Зажмите клавишу вызова радиального меню.
   - Удерживая клавишу, наведите мышь на нужную стратагему.
   - Отпустите клавишу, чтобы выполнить макрос.

### Важные примечания:

    Режим окна: Для корректного отображения меню игра должна быть в режиме «Оконный» или «Оконный без рамки».
    Права администратора: Если Steam или Helldivers 2 запущены от имени администратора, скрипт также необходимо запускать от имени администратора.
    Раскладка клавиатуры: Скрипт стабильнее всего работает с английской раскладкой. Если возникли проблемы, отредактируйте файл Settings.ini или просто удалите его (это сбросит все настройки), после чего перезапустите скрипт.

### Горячие клавиши по умолчанию:

    Средняя кнопка мыши — Радиальное меню
    F1 — Показать/скрыть основное окно (GUI)
    F2 — Показать/скрыть плавающий список профилей клавиш
    F3 — OCR: Сканирование стратагем
    F4 — Переключатель функции «Обход глушилки» (Scrambler Bypass)
    Insert — Приостановить/возобновить работу скрипта
    End — Закрыть скрипт
    Page Up / Page Down — Переключение профилей вперед/назад (работает только при удержании клавиши радиального меню или плавающего списка)

### Описание функции (*) Checkbox:

Добавляет оператор * к вашему хоткею. Это означает, что он будет срабатывать независимо от того, зажаты ли другие клавиши-модификаторы (Ctrl, Alt, Shift или Win).

### Обозначения AHK:

    LButton — Левая кнопка мыши
    RButton — Правая кнопка мыши
    MButton — Средняя кнопка мыши (колесо)
    XButton1 / XButton2 — Боковые кнопки мыши (4-я и 5-я)
    WheelUp / WheelDown — Прокрутка колеса вверх / вниз
    ^ — Клавиша Ctrl
    ! — Клавиша Alt
    + — Клавиша Shift
    * — Оператор «любой модификатор». Делает хоткей универсальным.
    ~ — Оператор «пропуска». Позволяет клавише выполнять своё стандартное игровое действие одновременно с активацией макроса.


### OCR (распознавание стратагем)

OCR позволяет автоматически определять доступные во время миссии стратагемы и использовать их через радиальное меню. Работа функции основана на распознавании последовательности стрелок каждой стратагемы.

    Сканирование стратагем (по умолчанию F3): После нажатия горячей клавиши, OCR считывает текущие стратагемы, автоматически создает OCR-профиль, переключается на него и заполняет его найденными стратагемами. После этого их можно сразу использовать через радиальное меню.
    Обход Scrambler (по умолчанию F4): При включенной функции каждый раз при открытии радиального меню выполняется поиск всех доступных стратагем. После выбора стратагемы OCR считывает текущую последовательность стрелок непосредственно с экрана и автоматически выполняет ввод.
    OCR для объектов задания: OCR также способен считывать и выполнять стратагемы, отображаемые непосредственно на интерфейсе, без открытия стандартного меню стратагем. Это относится к таким объектам задания, как поднятие флага, адская бомба, загрузка данных и другим подобным заданиям.

Замечания по производительности OCR: Работа OCR очень чувствительна к расположению и внешнему виду меню стратагем. Любые отклонения от стандартных настроек игры могут ухудшить качество распознавания или полностью нарушить его работу. Например: включение визуального эффекта Curved или изменение масштаба HUD меню стратагем. Разницу масштаба HUD можно компенсировать с помощью параметра In-Game HUD Scale в настройках OCR.

Если возникают проблемы с распознаванием: включите Debug Mode. На экране появится сетка распознавания стрелок. Проверьте, совпадает ли она с положением стрелок стратагем. Если сетка смещена, вручную измените ее положение до полного совпадения.

Методы распознавания:

1. Цветовой OCR (Color-Based OCR)

Данный метод определяет стрелки по их цвету. Следует учитывать, что он чувствителен к изменению цветов изображения. В очень ярких областях карты точность распознавания может снижаться. Если используется ReShade или другое программное обеспечение, изменяющее цветокоррекцию изображения, OCR может полностью перестать распознавать стрелки. В этом случае необходимо: включить Debug Mode и во время миссии выполнить OCR-сканирование (F3). В результате сканирование получите новый HEX-код цвета стрелок, заменить значение ArrowColor в настройках OCR.

2. OCR по форме (Shape-Based OCR)

Этот метод использует библиотеку FindText и определяет стрелки по их форме. На данный момент база шаблонов поддерживает только разрешения: 1920×1080 (1080p), 2560×1440 (1440p), 3840×2160 (2160p). Количество шаблонов пока относительно небольшое. Поэтому в большинстве случаев данный метод не обеспечивает более высокую точность по сравнению с цветовым OCR.
